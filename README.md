# LeetCode-Book-JS-and-TS

- 基础知识：时间复杂度和空间复杂度的概念和推算
- 数据结构：数组、栈（单调栈）、队列（双端队列）、字符串、链表、哈希表、树、图、堆
- 基本算法：查找、排序、双指针、滑动窗口、模拟法、数学、贪心算法、动态规划、分治、搜索与回溯
- 主要语言：Javascript、Typescript
